📱 TestFlight 설정 가이드

=== 1단계: App Store Connect에서 테스터 추가 ===

현재 위치: https://appstoreconnect.apple.com/apps/6753962694/testflight/ios

🔹 방법 A: 내부 테스팅 (빠름)
1. 왼쪽 메뉴 "TestFlight" → "내부 테스팅" 클릭
2. "App Store Connect 사용자" 섹션에서 본인 계정 확인
3. 본인 이메일이 없으면 "+ 내부 테스터" 클릭하여 추가
4. "빌드" 탭에서 1.0.0 빌드를 테스트 그룹에 추가

🔹 방법 B: 외부 테스팅 (권장)
1. "외부 테스팅" 클릭
2. "+ 그룹 추가" 또는 기존 그룹 선택
3. "테스터 추가" 클릭
4. iPhone Apple ID 이메일 입력
5. 빌드 1.0.0 선택
6. "저장 및 초대 전송"

=== 2단계: iPhone에서 확인 ===

1. 📧 이메일 확인 (TestFlight 초대 메일)
2. 📱 TestFlight 앱 열기
3. 🔍 "ArtYard" 또는 초대 목록 확인
4. 📲 "설치" 버튼 클릭
5. 🚀 앱 실행 및 테스트

=== 3단계: 스크린샷 촬영 ===

iPhone 8 Plus: 홈 버튼 + 사이드 버튼 동시 클릭

필요한 스크린샷:
📸 1. 홈 피드 (작품들 보이는 화면)
📸 2. 작품 상세 (큰 이미지 + 정보)
📸 3. 채팅 화면 (메시지 대화)
📸 4. 프로필 화면 (사용자 정보)
📸 5. 업로드 화면 (새 작품 올리기)
📸 6. 검색/필터 화면

=== 문제 해결 ===

❓ 테스터 목록에 안 보임: 이메일 주소 재확인
❓ 앱이 TestFlight에 안 보임: 빌드가 그룹에 할당되었는지 확인
❓ 설치 버튼 없음: 초대 상태가 "수락됨"인지 확인
