📱 카카오 iOS 앱 설정 가이드

=== 1단계: 카카오 개발자 콘솔 접속 ===
🌐 https://developers.kakao.com/console/app

=== 2단계: 플랫폼 설정 ===
📍 내 애플리케이션 > ArtYard > 앱 설정 > 플랫폼

✅ iOS 플랫폼 추가/수정:
- 번들 ID: com.artyard.app
- 앱스토어 ID: 6753962694 (App Store Connect에서 확인)
- 팀 ID: ${EXPO_PUBLIC_APPLE_TEAM_ID} (app.json에서 확인)

=== 3단계: 카카오 로그인 설정 ===
📍 제품 설정 > 카카오 로그인 > Redirect URI

✅ Redirect URI 추가:
1. https://bkvycanciimgyftdtqpx.supabase.co/auth/v1/callback
2. com.artyard.app://auth/callback (네이티브용)
3. exp://localhost:8081/--/auth/callback (개발용)

=== 4단계: 동의항목 설정 ===
📍 제품 설정 > 카카오 로그인 > 동의항목

✅ 필수 동의항목만 설정:
- 닉네임: 필수 (선택 해제 불가능하게 설정)
- 프로필 사진: 선택 (선택 해제 가능하게 설정)
- 카카오계정(이메일): 사용 안함 (비활성화)

⚠️ 중요: 이메일 동의항목을 비활성화해야 KOE101 에러가 해결됩니다!

=== 5단계: 앱 키 확인 ===
📍 앱 설정 > 앱 키

✅ JavaScript 키 확인:
- JavaScript 키를 Supabase OAuth 설정에 사용
- REST API 키는 네이티브 SDK용

=== 6단계: 보안 설정 ===
📍 제품 설정 > 카카오 로그인 > 보안

✅ Client Secret 사용 설정:
- Client Secret 코드 생성
- Supabase에서 카카오 OAuth 설정시 사용

=== 문제 해결 ===
❌ KOE101 에러: 동의항목에서 이메일 비활성화
❌ KOE006 에러: Redirect URI 확인
❌ 로그인 창이 안 뜸: 플랫폼 설정 확인
