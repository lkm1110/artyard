# Artyard 투자제안서

## Executive Summary

**Artyard**는 신진 작가와 아트 컬렉터를 연결하는 글로벌 아트 마켓플레이스입니다. 작가에게는 공정한 수익 분배(직거래 0%, 에스크로 10%)를, 구매자에게는 챌린지 시스템을 통한 검증된 작품 구매 경험을 제공합니다.

**투자 하이라이트**:
- **투자 요청**: Pre-Seed $500K - $1M (Post-money $5M)
- **목표 시장**: 글로벌 온라인 아트 마켓 $13.3B (연 9.8% 성장)
- **3년 목표**: GMV $30M, 작가 20,000명
- **차별화**: 작가 직거래 0% 수수료 + 챌린지/경매 시스템

---

## 1. Problem (문제점)

### 1.1 신진 작가들의 어려움

#### 높은 진입장벽
- 기존 갤러리: **40-60% 수수료**
- 작품 전시비: $2,000 - $5,000
- 작가 검증 부재로 신뢰 구축 어려움

#### 제한적인 판매 채널
- 오프라인 갤러리: 지역적 제약
- 기존 온라인 플랫폼: 높은 수수료 (20-30%)
- 글로벌 진출: 결제/정산 인프라 부족

#### 마케팅 비용
- 개인 SNS 마케팅: 시간과 비용 소모
- 광고 비용: 월 $500 - $2,000
- 효과적인 노출 기회 부족

### 1.2 구매자들의 문제

#### 신뢰 문제
- 작가 검증 부재 (학력, 경력 확인 불가)
- 작품 진위 확인 어려움
- 리뷰/평가 시스템 부족

#### 가격 투명성 부족
- 갤러리 마진이 숨겨져 있음
- 작품 가격 책정 기준 불명확
- 재판매 시 가치 예측 어려움

#### 발견의 어려움
- 신진 작가 발굴 채널 부족
- 큐레이션 부재
- 개인 취향 맞춤 추천 없음

## 2. Solution (솔루션)

### 2.1 작가 친화적 수수료 구조

#### 직거래 모드 (0% 수수료)
```
작품 가격: $500
작가 수익: $500 (100%)
플랫폼 수수료: $0

→ 작가가 직접 구매자와 소통하고 결제 링크 생성
→ 플랫폼은 작품 노출만 제공
```

#### 에스크로 모드 (10% 수수료)
```
작품 가격: $500
플랫폼 수수료: $50 (10%, 결제 수수료 포함)
작가 수익: $450 (90%)

→ 안전한 거래 보장 (에스크로)
→ 작가 신뢰도 점수 상승
→ 작품 배송 추적
→ 구매자 보호 (환불 정책)
```

**초기 작가 혜택**: 첫 3건 거래 시 에스크로 수수료 50% 할인 (5%)

**경쟁사 비교**:
| 플랫폼 | 수수료 | 작가 수익 |
|--------|--------|-----------|
| Artyard (직거래) | 0% | 100% |
| Artyard (에스크로) | 10% | 90% |
| Artyard (초기 작가) | 5% | 95% |
| Saatchi Art | 35% | 65% |
| Artfinder | 33% | 67% |
| Etsy | 20% | 80% |

### 2.2 챌린지 시스템

#### 작가 발굴 & 검증
- 2주간 테마 챌린지
- 커뮤니티 투표 (1계정 1표)
- Top 3 선정 및 노출

#### 구매자 참여
- 투표를 통한 작품 큐레이션
- 우승작 우선 구매권
- 커뮤니티 형성

#### 마케팅 효과
- 챌린지 참여 = 무료 홍보
- SNS 바이럴 효과
- 플랫폼 트래픽 증가

### 2.3 분기별 경매 시스템

#### 우승작 프리미엄 판매
```
챌린지 우승작 → 분기별 경매 출품
- 경매 시작가: 작가 설정 (Reserve Price)
- 입찰 방식: 실시간 경매
- Buy Now 옵션: 시작가의 5배
- 플랫폼 수수료: 10%
```

#### 수익 구조
```
경매 낙찰가: $1,000
작가 수익: $900 (90%)
플랫폼 수수료: $100 (10%)

→ 작가는 높은 가격에 판매
→ 플랫폼은 프리미엄 작품으로 브랜드 강화
```

### 2.4 작가 신뢰도 시스템

#### Tier System
| Tier | 조건 | 혜택 |
|------|------|------|
| New | 가입 후 | 챌린지 참여 가능 |
| Trusted | 거래 3건 + 평점 4.0+ | 프로필 뱃지, 우선 노출 |
| Verified | 거래 10건 + 평점 4.5+ | 신뢰 점수 표시, 추천 |
| Pro | 거래 50건 + 평점 4.8+ | 전용 매니저, 글로벌 마케팅 |

#### 자동 업그레이드
- 거래 완료 시 자동 계산
- 평점 자동 집계
- 알림으로 축하 메시지

### 2.5 글로벌 결제/정산

#### 2Checkout 통합
- 200개국 이상 지원
- 87개 통화 지원
- 자동 환전 및 정산
- Split Payment (작가/플랫폼 자동 분배)

#### KYC/신원 확인
- 2Checkout KYC 시스템
- 여권/신분증 자동 검증
- 글로벌 컴플라이언스 준수
- 반자동 정산 (월 1회)

## 3. Market Opportunity (시장 기회)

### 3.1 시장 규모

#### 글로벌 아트 시장
- **총 시장 규모**: $65B (2024)
- **온라인 시장**: $13.3B (2024)
- **성장률**: 연 9.8%
- **2030년 예상**: $25B

#### 온라인 아트 구매 증가
- 2019년: $4.8B
- 2020년: $12.4B (팬데믹, 159% 증가)
- 2024년: $13.3B
- 밀레니얼/Z세대: 온라인 구매 선호 85%

### 3.2 타겟 시장

#### Primary (1차 시장)
**작가**: 신진 작가, 아마추어 작가
- 타겟: 미술 전공 학생, 취미 작가
- 규모: 글로벌 1,000만명 이상
- 연령: 20-35세
- 지역: 글로벌 (전세계 동시 오픈)

**구매자**: 밀레니얼/Z세대 컬렉터
- 타겟: 20-40세, 온라인 쇼핑 선호
- 구매력: $300 - $3,000/작품
- 관심사: 인테리어, 독특한 작품, 투자

#### Secondary (2차 시장)
**갤러리/큐레이터**
- 타겟: 소규모 갤러리, 프리랜스 큐레이터
- 니즈: 신진 작가 발굴

**기업 구매자**
- 타겟: 스타트업, 디자인 스튜디오
- 니즈: 사무실 아트, 대량 구매

### 3.3 경쟁 분석

#### 직접 경쟁사
| 플랫폼 | 강점 | 약점 |
|--------|------|------|
| Saatchi Art | 브랜드, 큐레이션 | 높은 수수료 (35%), 미국 중심 |
| Artfinder | 글로벌, 작가 커뮤니티 | 높은 수수료 (33%) |
| Etsy | 거대한 사용자 기반 | 작품 품질 낮음, 아트 전문성 부족 |
| 인스타그램 | 무료, 거대한 노출 | 결제/정산 없음, 신뢰 부족 |
| Artsy | 300만+ 사용자, 갤러리 파트너 | 갤러리 중심 (B2B2C), 높은 진입장벽 |

#### Artyard의 경쟁 우위
1. **가격 경쟁력**: 직거래 0%, 에스크로 10% (업계 최저) + 초기 작가 50% 할인
2. **챌린지 시스템**: 작가 발굴 + 구매자 참여 + 바이럴 마케팅
3. **경매 시스템**: 프리미엄 작품 판매 채널 (분기별)
4. **글로벌 인프라**: 2Checkout 기반 200개국 지원, 처음부터 전세계 오픈
5. **작가 신뢰도**: 자동 검증 및 티어 시스템
6. **다국어 지원**: 영문 기반, 향후 한국어/일본어/중국어 추가 예정
7. **작가 중심**: Artsy의 갤러리 중심과 달리, 작가가 직접 판매

## 4. Business Model (비즈니스 모델)

### 4.1 수익 구조

#### Revenue Streams

**1. 일반 작품 판매 - 에스크로 거래 수수료 (10%)**
```
예상 GMV: Year 1 $500K
에스크로 비율: 60%
수익: $500K × 60% × 10% = $30K

→ 핵심 수익원: 작가와 구매자 간 안전한 거래 중개
→ 결제 수수료 포함 (2Checkout 통합)
```

**2. 챌린지 & 경매 시스템 (10%)**
```
분기별 경매: 4회/년
작품당 평균 낙찰가: $1,000
작품 수: 12개/년
수익: 12 × $1,000 × 10% = $1.2K (Year 1)

→ 수익보다는 마케팅 효과 및 플랫폼 흥행 요소
→ 신진 작가 발굴 및 브랜드 강화
→ 커뮤니티 참여 유도
```

**3. 환전 수수료 (향후, Year 2+)**
```
직거래 작품 판매 시 환전 서비스 제공
환차익 수수료: 1-2%
예상 GMV (직거래): Year 2 $2M
예상 수익: $20K - $40K

→ 직거래 0% 수수료 유지하면서도 수익 창출
→ 87개 통화 지원 활용
→ 작가들에게 편리한 환전 서비스 제공
```

**4. 광고 수익 (향후, Year 3+)**
```
갤러리 광고, 아트 용품 브랜드
예상 수익: $20K (Year 3)
```

### 4.2 Unit Economics

#### 작가 1명당 수익
```
가정:
- 작품 업로드: 평균 5개/년
- 판매율: 20%
- 평균 작품 가격: $500
- 에스크로 사용률: 60%

연간 GMV/작가: 5 × 20% × $500 = $500
플랫폼 수익/작가: $500 × 60% × 10% = $30

작가 획득 비용 (CAC): $50 (마케팅 + 온보딩)
LTV: $30 × 3년 = $90
LTV/CAC: 1.8x (Year 1) → 3x (Year 3)
```

#### 구매자 1명당 수익
```
가정:
- 연간 구매: 2회
- 평균 구매 금액: $400
- 에스크로 사용률: 80%

연간 GMV/구매자: 2 × $400 = $800
플랫폼 수익/구매자: $800 × 80% × 10% = $64

구매자 획득 비용 (CAC): $20 (마케팅)
LTV: $64 × 5년 = $320
LTV/CAC: 16x
```

## 5. Growth Strategy (성장 전략)

### 5.1 작가 획득 전략

#### Phase 1: 초기 시드 (0-50명, 1-3개월)
- 글로벌 미술대학 온라인 커뮤니티 접근
- 인스타그램 아티스트 DM (팔로워 1K-10K)
- 초기 작가 인센티브: 첫 3건 거래 수수료 50% 할인
- 예산: $10K

#### Phase 2: 커뮤니티 성장 (50-500명, 4-9개월)
- 첫 챌린지 개최 (상금 $1,000)
- 글로벌 아트 인플루언서 협업 (10명)
- 작가 추천 프로그램 (추천당 $50)
- 예산: $50K

#### Phase 3: 스케일업 (500-5,000명, 10-24개월)
- Facebook/Instagram/TikTok 광고
- 글로벌 아트 스쿨 파트너십 (20개 학교)
- 챌린지 확대 (월 2회)
- 예산: $200K

### 5.2 구매자 획득 전략

#### SEO & 콘텐츠 마케팅
- 작가 인터뷰 블로그 (주 2회, 영문)
- 아트 가이드 콘텐츠
- "emerging artists" 키워드 타겟

#### SNS 마케팅
- 인스타그램: 작품 사진 + 작가 스토리
- 틱톡: 챌린지 하이라이트, 작품 제작 과정
- 유튜브: 작가 다큐멘터리

#### 커뮤니티 마케팅
- 챌린지 투표 참여 유도
- 구매 후 리뷰 인센티브
- 친구 초대 프로그램

#### 페이드 마케팅
- Google Ads: "buy art online", "emerging artists" 키워드
- Facebook/Instagram: 25-40세, 인테리어/아트 관심
- 예산: $10K/월 (Year 2부터)

### 5.3 글로벌 확장 전략

#### Phase 1: 글로벌 동시 런칭 (Q1 2026)
- **영문 기반 플랫폼** (전세계 오픈)
- 2Checkout 기반 200개국 결제 지원
- 87개 통화 지원
- 글로벌 작가 동시 모집

#### Phase 2: 다국어 확장 (Q3-Q4 2026)
- 한국어, 일본어 추가
- 아시아 마케팅 강화
- 현지 아트 커뮤니티 파트너십

#### Phase 3: 전방위 확장 (2027)
- 중국어, 스페인어, 프랑스어 추가
- 유럽/남미 현지 마케팅
- 글로벌 아트 스쿨 파트너십
- 200개국 전체 활성화
