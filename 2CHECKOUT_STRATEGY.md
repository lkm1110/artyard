## 📋 **사업자 등록 가이드**

### **개인 사업자 (추천)** ⭐

```
장점:
✅ 빠름 (온라인 1시간 + 승인 1-2주)
✅ 간단 (서류 적음)
✅ 저렴 (무료)

과정:
1. 정부24 접속 (www.gov.kr)
2. "사업자등록 신청" 검색
3. 온라인 신청:
   - 업종: 전자상거래업
   - 사업장: 자택 가능
4. 서류 업로드:
   - 신분증
   - 임대차계약서 (or 등기부등본)
5. 1-2주 후 사업자 번호 발급
6. 통장 개설

비용: 무료
시간: 2-3주
```

### **법인 사업자**

```
장점:
✅ 신뢰도
✅ 투자 유치 유리

단점:
❌ 복잡
❌ 비용 (50-100만원)
❌ 시간 (4-6주)

→ 나중에 필요하면 전환 가능
```
