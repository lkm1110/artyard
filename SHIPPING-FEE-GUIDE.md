# 🚚 배송비 처리 완벽 가이드

## 🎯 핵심 답변

### **Q: 글로벌 서비스인데 배송비는 누가 부담하나요?**

**A: 구매자가 부담합니다!** ✅

```
작품 가격: 50,000원 (판매자 설정)
+ 배송비: 3,000원 (구매자 부담)
━━━━━━━━━━━━━━━━━━━━━━━━━━
총 결제액: 53,000원 (구매자 결제)

판매자 수령: 43,250원
  (50,000원 - 플랫폼 수수료 - Stripe 수수료)

→ 배송비는 플랫폼이 받아서 택배사에 지불
→ 판매자는 작품 가격만 신경 쓰면 됨
```

## 📊 글로벌 마켓플레이스 비교

### **누가 배송비를 내나?**

```
Etsy: 구매자 부담 ✅
eBay: 구매자 부담 ✅
Amazon: 구매자 부담 (프라임은 무료)
Depop: 구매자 부담 ✅
Mercari: 구매자 부담 ✅
Artsy: 작품별 다름 (대부분 구매자)

→ 99%가 구매자 부담!
```

### **왜 구매자가 내나요?**

```
1. 공평함 ⚖️
   - 배송지가 멀면 비용 증가
   - 배송 옵션 선택권은 구매자에게
   
2. 투명함 💎
   - 작품 가격 따로
   - 배송비 따로
   - 명확한 비용 구조
   
3. 판매자 보호 🛡️
   - 판매자는 작품 가격만 설정
   - 배송비로 손해 안 봄
   - 국제 배송도 부담 없음
```

## 🌍 국내 vs 국제 배송비

### **국내 배송 (한국 → 한국)**

```typescript
┌──────────────────────────┐
│ 배송 옵션 선택            │
├──────────────────────────┤
│ ⭕ 일반 배송 (2-3일)      │
│    배송비: 3,000원        │
│    택배사: 우체국/CJ      │
│                          │
│ ⭕ 빠른 배송 (1일)        │
│    배송비: 5,000원        │
│    택배사: CJ대한통운     │
│                          │
│ 💡 10만원 이상 무료 배송! │
└──────────────────────────┘

실제 비용:
- 우체국 택배: 3,000원
- CJ대한통운: 3,500원
- 로젠택배: 2,800원

→ 플랫폼 수익: 0-500원
```

## 📝 DB 스키마 (배송비 관련)

```sql
-- 작품별 배송 설정
CREATE TABLE artwork_shipping_settings (
  id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  artwork_id UUID NOT NULL REFERENCES artworks(id) ON DELETE CASCADE,
  
  -- 국내 배송
  domestic_enabled BOOLEAN DEFAULT true,
  domestic_fee INTEGER DEFAULT 3000, -- 배송비 (원)
  domestic_free_threshold INTEGER DEFAULT 100000, -- 무료 배송 기준
  
  -- 국제 배송
  international_enabled BOOLEAN DEFAULT false,
  asia_fee INTEGER DEFAULT 12000,
  north_america_fee INTEGER DEFAULT 18000,
  europe_fee INTEGER DEFAULT 25000,
  
  -- 특수 사항
  fragile BOOLEAN DEFAULT false, -- 파손 주의
  requires_signature BOOLEAN DEFAULT false, -- 서명 필수
  
  created_at TIMESTAMPTZ DEFAULT NOW(),
  updated_at TIMESTAMPTZ DEFAULT NOW()
);

-- 거래에 배송비 기록
ALTER TABLE transactions 
ADD COLUMN shipping_fee INTEGER NOT NULL DEFAULT 0,
ADD COLUMN shipping_zone VARCHAR(50) DEFAULT 'domestic', -- 'domestic', 'asia', 'north_america', 'europe'
ADD COLUMN shipping_option VARCHAR(50) DEFAULT 'standard'; -- 'standard', 'express'

-- 예시 데이터
INSERT INTO artwork_shipping_settings (artwork_id, domestic_enabled, international_enabled)
VALUES 
  ('abc-123', true, false), -- 국내만
  ('def-456', true, true);  -- 국제 허용
```
